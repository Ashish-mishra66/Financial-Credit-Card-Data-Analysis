## Project Overview
This project focuses on analyzing **credit card financial data** to understand **revenue patterns, customer income, and expense behavior** across multiple factors.  
The analysis was performed using **Power BI** to create interactive dashboards that help in identifying trends and insights useful for business decision-making.

---

## Objectives
- Analyze **credit card revenue** across different customer segments  
- Understand **income vs expenses** patterns of customers  
- Identify key factors influencing revenue generation  
- Build an **interactive Power BI dashboard** for clear visualization  

---

## Tools & Technologies Used
- Power BI  
- SQL  
- Microsoft Excel  

---

## Key Analysis & Insights
- Revenue distribution across different customer categories  
- Monthly and yearly credit card revenue trends  
- Comparison between **customer income and expenses**  
- Identification of high-value and low-value customers  
- Expense behavior impact on overall credit card revenue  


## Data Description
The dataset includes:
- Customer demographic information  
- Credit card transaction data  
- Income and expense details  
- Time-based revenue records  

---

## Business Value
This dashboard helps:
- Financial institutions understand **customer spending behavior**  
- Identify profitable customer segments  
- Improve financial planning and revenue strategies  

---

## Conclusion
The project demonstrates how **Power BI dashboards** can convert raw financial data into meaningful insights, enabling data-driven decision-making in the finance domain.
